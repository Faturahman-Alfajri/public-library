# Public Library Management Information System (Campus)

A public-facing prototype for a campus library portal.  
Main features: a searchable book catalog (dummy JSON) and a service complaint form saved to Google Sheets via Google Apps Script.  
Deployed on GitHub Pages (accessible anywhere).

## Features
- Book Catalog: search/filter, availability status, and shelf location
- Service Complaint: submit complaints and store them automatically in Google Sheets (Apps Script)
- Clean responsive UI using Bootstrap

## Tech Stack
- HTML, CSS, JavaScript
- Bootstrap 5
- GitHub Pages (deployment)
- Google Sheets + Google Apps Script (complaint storage)
- Dummy JSON dataset (catalog)

## How to Run Locally
1. Download/clone the repository
2. Open `index.html` in a browser  
   (or run with a local server like XAMPP for development)

## Notes
- The catalog uses dummy data for presentation.
